webpack з¬”и®°

з¬¬ 1 з« пјҡwebpack з®Җд»Ӣ

д»Җд№ҲжҳҜwebpack?

е®ҳж–№дёӢе®ҡд№үпјҡжң¬иҙЁдёҠпјҢwebpack жҳҜдёҖдёӘзҺ°д»Ј JavaScript еә”з”ЁзЁӢеәҸзҡ„йқҷжҖҒжЁЎеқ—жү“еҢ…еҷЁ(module bundler)гҖӮеҪ“ webpack еӨ„зҗҶеә”з”ЁзЁӢеәҸж—¶пјҢе®ғдјҡйҖ’еҪ’ең°жһ„е»әдёҖдёӘдҫқиө–е…ізі»еӣҫ(dependency graph)пјҢе…¶дёӯеҢ…еҗ«еә”з”ЁзЁӢеәҸйңҖиҰҒзҡ„жҜҸдёӘжЁЎеқ—пјҢз„¶еҗҺе°ҶжүҖжңүиҝҷдәӣжЁЎеқ—жү“еҢ…жҲҗдёҖдёӘжҲ–еӨҡдёӘ bundleгҖӮ

webpackеҸҜд»Ҙе°ҶжҢүйңҖеҠ иҪҪзҡ„жЁЎеқ—иҝӣиЎҢд»Јз ҒеҲҶзҰ»пјҢзӯүеҲ°е®һйҷ…йңҖиҰҒж—¶еҶҚејӮжӯҘеҠ иҪҪгҖӮйҖҡиҝҮloaderиҪ¬жҚўпјҢд»»дҪ•еҪўејҸзҡ„иө„жәҗйғҪеҸҜд»ҘеҪ“еҒҡжЁЎеқ—пјҢжҜ”еҰӮCommonsJSгҖҒAMDгҖҒES6гҖҒ CSSгҖҒJSONгҖҒCoffeeScriptгҖҒLESSзӯү;

webpackдёӢиҪҪ

еүҚжҸҗжқЎд»¶

еңЁејҖе§Ӣд№ӢеүҚпјҢиҜ·зЎ®дҝқе®үиЈ…дәҶ Node.js зҡ„жңҖж–°зүҲжң¬гҖӮдҪҝз”Ё Node.js жңҖж–°зҡ„й•ҝжңҹж”ҜжҢҒзүҲжң¬(LTS - Long Term Support)пјҢжҳҜзҗҶжғізҡ„иө·жӯҘгҖӮдҪҝз”Ёж—§зүҲжң¬пјҢдҪ еҸҜиғҪйҒҮеҲ°еҗ„з§Қй—®йўҳпјҢеӣ дёәе®ғд»¬еҸҜиғҪзјәе°‘ webpack еҠҹиғҪд»ҘеҸҠ/жҲ–иҖ…зјәе°‘зӣёе…і package еҢ…гҖӮ

еҰӮжһңдҪ дёҚжё…жҘҡдҪ дёӢиҪҪзҡ„nodejsжҳҜеҗҰжҳҜжңҖж–°зҡ„зүҲжң¬пјҢжң¬ж–ҮдёӢж–№жңүиҜҰз»Ҷи®Іи§ЈгҖӮ

дёӢиҪҪж–№ејҸпјҡжү“ејҖcmd,cdеҲ°дҪ жғіиҰҒеӮЁеӯҳзҡ„ж–Үд»¶зӣ®еҪ•пјҢиҫ“е…Ҙд»ҘдёӢе‘Ҫд»Өе…¶дёӯдёҖжқЎпјҡ

еүҚдёҖжқЎе‘Ҫд»ӨдёәпјҡwebpackжңҖж–°зүҲжң¬пјӣеҗҺдёҖжқЎе‘Ҫд»Өдёәпјҡwebpackзү№е®ҡзүҲжң¬гҖӮ

зӨәдҫӢеӣҫпјҡ

nodejsдёӢиҪҪе®ҳж–№дёӢиҪҪй“ҫжҺҘпјҡnodejsдёӢиҪҪ

иҮідәҺдёӢиҪҪзҡ„йҖүжӢ©дёҠе»әи®®йҖүжӢ©LTSпјҲx64пјүпјҢLTSжҳҜй•ҝжңҹзЁіе®ҡзүҲжң¬гҖӮ

жЈҖжҹҘnodejsзүҲжң¬еҸ·

еңЁwindowsдёҠжҹҘзңӢnodejsзҡ„зүҲжң¬еҸ·

  • 1гҖҒеңЁејҖе§ӢиҸңеҚ•зҡ„жҗңзҙўжЎҶдёӯиҫ“е…ҘcmdпјҢзӮ№еҮ»cmd.exeжү“ејҖcmdе‘Ҫд»ӨзӘ—еҸЈжҲ–иҖ…дҪҝз”Ёеҝ«жҚ·й”®windows+RпјҢиҫ“е…ҘcmdпјҢжҢүдёӢеӣһиҪҰгҖӮ

  • 2гҖҒдҪҝз”Ёе‘Ҫд»ӨжҹҘзңӢnode.jsзҡ„зүҲжң¬еҸ·

еңЁcmdе‘Ҫд»ӨиЎҢдёӯдҪҝз”Ёcdе‘Ҫд»Өиҝӣе…Ҙnodejsе®үиЈ…зӣ®еҪ• пјҢдҫӢеҰӮпјҡ

жҺҘзқҖеҶҚиҫ“е…Ҙ node -vе‘Ҫд»ӨпјҢе°ұеҸҜжҹҘзңӢеҲ°nodeзҡ„зүҲжң¬еҸ·пјҢдҫӢеҰӮжҲ‘зҡ„зүҲжң¬еҸ·е°ұжҳҜv16.14.2гҖӮ

webpack дә”дёӘж ёеҝғжҰӮеҝө

1. Entry

вҖӢ е…ҘеҸЈ(Entry)жҢҮзӨә webpack д»Ҙе“ӘдёӘж–Үд»¶дёәе…ҘеҸЈиө·зӮ№ејҖе§Ӣжү“еҢ…пјҢеҲҶжһҗжһ„е»әеҶ…йғЁдҫқиө–еӣҫгҖӮ

2. Output

вҖӢ иҫ“еҮә(Output)жҢҮзӨә webpack жү“еҢ…еҗҺзҡ„иө„жәҗ bundles иҫ“еҮәеҲ°е“ӘйҮҢеҺ»пјҢд»ҘеҸҠеҰӮдҪ•е‘ҪеҗҚгҖӮ

3. Loader

вҖӢ Loader и®© webpack иғҪ еӨҹ еҺ» еӨ„ зҗҶ йӮЈ дәӣ йқһ JavaScript ж–Ү д»¶ (webpack иҮӘ иә« еҸӘ зҗҶ и§ЈJavaScript)

4. Plugins

вҖӢ жҸ’д»¶(Plugins)еҸҜд»Ҙз”ЁдәҺжү§иЎҢиҢғеӣҙжӣҙе№ҝзҡ„д»»еҠЎгҖӮжҸ’д»¶зҡ„иҢғеӣҙеҢ…жӢ¬пјҢд»Һжү“еҢ…дјҳеҢ–е’ҢеҺӢзј©пјҢдёҖзӣҙеҲ°йҮҚж–°е®ҡд№үзҺҜеўғдёӯзҡ„еҸҳйҮҸзӯүгҖӮ

5. Mode

вҖӢ жЁЎејҸ(Mode)жҢҮзӨә webpack дҪҝз”Ёзӣёеә”жЁЎејҸзҡ„й…ҚзҪ®гҖӮ

йҖүйЎ№ жҸҸиҝ° зү№зӮ№
development дјҡе°Ҷ DefinePlugin дёӯ process.env.NODE_ENV зҡ„еҖји®ҫзҪ®дёә developmentгҖӮеҗҜз”Ё NamedChunksPlugin е’Ң NamedModulesPluginгҖӮ иғҪи®©д»Јз Ғжң¬ең°и°ғиҜ•иҝҗиЎҢзҡ„зҺҜеўғ
production дјҡе°Ҷ DefinePlugin дёӯ process.env.NODE_ENV зҡ„еҖји®ҫзҪ®дёә productionгҖӮ еҗҜз”ЁFlagDependencyUsagePlugin,FlagIncludedChunksPlugin, ModuleConcatenationPlugin, NoEmitOnErrorsPlugin, OccurrenceOrderPlugin, SideEffectsFlagPlugin е’Ң TerserPluginгҖӮ иғҪи®©д»Јз ҒдјҳеҢ–дёҠзәҝиҝҗиЎҢзҡ„зҺҜеўғ
